# Installation

### pip
```
pip install imdb-wiki-dataset
```


### Development Installation

* Clone the project.
* Install in Anaconda3 environment
```
$ conda env create --force -f dev_environment.yml
$ source activate imdb-wiki-dataset
$ pip install -e .
```
